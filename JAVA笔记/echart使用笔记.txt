查看API/配置项文档顺序

2：根据类型顺藤摸瓜一步步配置


注意：series 可以设置多个，用数组形式配置
例如： series : [
	        {
	            name:'出口',
	            type:'bar',
	            data:[1209, 1009, 17567, 1010, 6549,1909,1509],
	            
	        },
	        {
	            name:'入口',
	            type:'bar',
	            data:[17567, 6549, 12567, 1310, 5549,1409,1209],
	            
	        }
	    ]
